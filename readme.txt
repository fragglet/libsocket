Introduction
------------

This beta has many changes & is still a little rough in place. Some parts
like the Unix domain socket support are definitely work-in-progress. Please
read the supplied text documents before using libsocket:

announce.txt
bugs.txt
changes.txt
credits.txt
install.txt
license.txt
relnotes.txt

You may also want to look in redist/ at the redistributable components of
libsocket. These should be distributed with any programs you write using
libsocket.

Documentation
-------------

The documentation supplied with the binary distribution is in info-format
only. The documentation distribution contains the manual in HTML, as well as
a few relevant MS Knowledge Base articles.
